# 🏦 Bank Credit Approval Prediction (AI/ML Project)

## 📌 Overview

This project predicts whether a credit application should be approved or rejected using machine learning techniques. It simulates a real-world banking decision system.

---

## 🚀 Features

* Data cleaning and preprocessing
* Handling missing values
* Categorical encoding
* Logistic Regression model
* Model evaluation (Accuracy, Precision, Recall, F1-score)
* Visualization using Seaborn & Matplotlib

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Seaborn, Matplotlib
* Jupyter Notebook

---

## 📊 Dataset

* UCI Credit Approval Dataset
* 690 records with financial and personal attributes

---

## 📈 Model Performance

* Accuracy: **86%**
* Balanced precision and recall for both classes

---

## 📌 Workflow

1. Data Loading
2. Data Cleaning
3. Missing Value Handling
4. Feature Encoding
5. Model Training
6. Model Evaluation
